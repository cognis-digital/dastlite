dastlite is a security-checking tool for websites and web APIs. You point it at a URL or a list of URLs, and it automatically inspects the responses for common security problems — things like missing privacy headers, cookies that could be stolen, or pages that accidentally expose sensitive information. It produces a report you can read in the terminal, save as JSON, or feed directly into GitHub's code-scanning dashboard. It is built for developers and security engineers who want a fast, no-fuss security check that fits into an automated build pipeline.
